# Gemini System Prompts — Personality Presets

March 2026.

## What These Are

These are **NOT** Gems (user-created custom personas). These are Google's A/B testing personality presets that define how Gemini introduces itself when a user first connects their Google data (Gmail, Photos, Search, Calendar, Chat). Each preset represents a different onboarding behavioral mode.
